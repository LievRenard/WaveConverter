# WaveConverter
Light wave quantity converter. Convert a quantity among wavelength, wavenumber, frequency and energy.

### Supported Units
- Wavelength: m, cm, mm, um, nm
- Frequency: Hz, GHz, THz, rad/s
- Wavenumber: cm^-1(cm-1), rad/m, rad/cm, rad/um
- Energy: eV

### Usage
> WaveConverter [value] [original unit] to [destination unit]

### Example
> $ WaveConverter 1240nm to eV
> 
> Input: 1240.000 nm
> Converted: 1.000 eV


> $ WaveConverter 21cm to GHz
>
> Input: 21.000 cm
> Converted: 1.428 GHz
